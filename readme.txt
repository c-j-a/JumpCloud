-------------------------------------------------------------------------------
Project Assumptions
-------------------------------------------------------------------------------
You have an IDE installed that supports Maven such as IntelliJ or Eclipse.
You have a Java JDK version of 1.8 or greater.


-------------------------------------------------------------------------------
IntelliJ
-------------------------------------------------------------------------------
Downloading
1. Go to https://www.jetbrains.com/idea/download
2. Choose your OS (Windows / macOS / Linux)
3. Click the DOWNLOAD button under Community
4. Install

Import Project
1. Start IntelliJ
2. Click the 'File' menu and then 'Open'
3. Navigate to the root directory of the project and click 'OK'
4. Click the 'Build' menu then 'Build Project'

Running Tests
1. Navigate to src/test/java/com/jc/test/action/service and open ActionServiceTest.java
2. Click the 'Run' menu and then 'Run' or 'Debug'
3. Choose 'ActionServiceTest'
4. Five tests should have been run and all five should pass


-------------------------------------------------------------------------------
Eclipse
-------------------------------------------------------------------------------
Downloading (Portable version)
1. Go to https://www.eclipse.org/downloads/packages/release/2019-03/r/eclipse-ide-enterprise-java-developers
2. Choose your OS under 'Download Links' anc click Download
3. Unzip to your directory of choice

Import Project
1. Start eclipse (Close the welcome screen if it is there)
2. Click the 'File' menu and then 'Open Projects from File System...'
3. Navigate to the root directory of the project and click 'Select Folder'
4. Click the 'Finish' button
5. The project should build automatically, if it didn't, build from the 'Project' menu.

Running Tests
1. Navigate to src/test/java/com/jc/test/action/service and open ActionServiceTest.java
2. Click the 'Run' menu and then 'Run As' or 'Debug As' and then 'JUnit Test'
3. Five tests should have been run and all five should pass
